# PLANM : 플랜매니저    
>PLANM은 무료 ERP 서비스입니다.     
PLANM은 일정관리, 메일, 근태관리, 전자결재 등의 서비스를 제공합니다.
-----
## ⚙ 환경
+ Spring framework
+ Spring Security
+ jdk 1.8
+ Tomcat 9.0
+ MariaDB

## 🛠 기술
+ jQuery, jQuery-UI, jQuery-MultiFile
+ MyBatis
+ Tiles

## 📙 오픈소스
+ Bootstrap 4.4
+ Fullcalendar.js
+ SweetAlerts
+ ax5grid
+ Ckeditor, CkFinder
