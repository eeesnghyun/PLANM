# PLANM
> PLANM은 무료 ERP 솔루션입니다.



## 환경
+ Spring framework
+ Spring Security
+ jdk 1.8
+ Tomcat 9.0
+ MariaDB

## 기술
+ jQuery, jQuery-UI, jQuery-MultiFile
+ MyBatis
+ Tiles

## 오픈소스
+ Bootstrap 4.4
+ Fullcalendar.js
+ SweetAlerts
+ ax5grid
+ Ckeditor, CkFinder
