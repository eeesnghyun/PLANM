# PLANM    
> PLANM은 일정, 사내 메일, 근태, 전자결재 서비스를 제공하는 무료 ERP 서비스입니다.    
-----
## 환경
+ Spring framework
+ Spring Security
+ jdk 1.8
+ Tomcat 9.0
+ MariaDB

## 기술
+ jQuery, jQuery-UI, jQuery-MultiFile
+ MyBatis
+ Tiles

## 오픈소스
+ Bootstrap 4.4
+ Fullcalendar.js
+ SweetAlerts
+ ax5grid
+ Ckeditor, CkFinder
